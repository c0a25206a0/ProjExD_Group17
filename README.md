# フェイクマリオ

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
* 主人公キャラクターを十字キーによりステージを攻略するゲームです

## ゲームの遊び方
* 矢印キーで主人公を操作し，スペースキー押下によるジャンプで障害物を越えます。
* 穴に落ちると最初からやり直しです

## ゲームの実装
### 共通基本機能
* 主人公の実装とステージの実装

### 分担追加機能
* 基本ステージ（担当：大橋　樹）：基本的なステージの作成
* デザイン（担当：榎本　優一）：主人公のデザイン、敵のデザイン、ゴールのデザイン
* サウンドエフェクト（担当：内山田　悠輝）：BGMと効果音
* アイテム（担当：米澤　璃琥）：主人公がパワーアップするアイテムを追加
* ボス（担当：）：ボスの追加
